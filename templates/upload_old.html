About

Can we use machine learning to assess whether novel sequences express cleanly in an organ of interest?

Because current gene therapies rely on fast-acting and messy promoter sequences to initiate expression, a major industry focus is designing novel promoter sequences that act locally in an organ of interest. However, assessing the expression of these novel sequences is performed in cell lines, which is costly and time-consuming. If we can predict whether a novel sequences is likely to be expressed in a single organ of interest, we can screen out ineffective sequences and accelerate the pace of drug discovery.

Using machine learning and leveraging publicly-available genetic sequence and expression data, I built a classification model to predict "liver-specific" expression or gene expression that is localized to the liver and not present in other organs (e.g. the brain). This application reads a common genetic sequence file type (FASTA files, *.fa) and calculates the probability that each sequence will be "liver-specific."
